What is Stoichiometry?

There are two types of stoichiometry, composition stoichiometry and reaction stoichiometry

  • Composition stoichiometry is a tool used to describe the numeric relationships between elements in compounds
  • Reaction stoichiometry is a tool used to describe the relationships of substances as they are undergoing chemical changes
